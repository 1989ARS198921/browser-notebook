# Участие в разработке browser-notebook

Спасибо за интерес к нашему проекту! Мы рады любому вкладу.

## Стандарты

*   **PEP 8:** Следуйте руководству по написанию кода для Python.
*   **PEP 263:** Добавляйте `# -*- coding: utf-8 -*-` в начало файлов `.py`, если они содержат не-ASCII символы.
*   **Commit Messages:** Используйте осмысленные сообщения коммитов на английском языке (желательно) или русском.

## Процесс

1.  Сделайте fork репозитория.
2.  Создайте новую ветку (`git checkout -b feature/my-new-feature`).
3.  Внесите изменения.
4.  Сделайте коммит (`git commit -am 'Add some feature'`).
5.  Запушьте ветку (`git push origin feature/my-new-feature`).
6.  Создайте Pull Request.
